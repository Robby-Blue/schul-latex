\section{Der genetische Drift}
Veränderungen der Genfrequenz aufgrund von zufälliger Ereignisse in kleinen Populationen werden \emph{genetischer Drift} genannt.
 
\subsection{Funktionsweise} 
Kommt es dazu, dass augrund von z.\,B. einer Umweltkatastrophe Zufällig verteilte Lebewesen sterben, so ist die Wahrscheinlichkeit größer, dass manche Genotypen überproportional und unterproportional häufig überleben. 
 
Dadurch wächst oder sinkt jeweils die Genfrequenz von genau dieser Gene.
