\section{Resonanz}
\emph{Resonanz} tritt auf, wenn die Schwingung des einen schwingungenden Systems die Schwingung eines anderen Verursacht. Dabei wird Energie vom \emph{Erreger} zum \emph{Resonator} übertragen. Wie viel Energie übertragen wird hängt von der Schwingungsfrequenz des Erregers und der Eigenfrequenz des Resonators ab, je ähnlicher diese sind, desto mehr Energie wird übertragen. Sind diese gleich, kommt es zu einer \emph{Resonanzkatastrophe}.
 
Andersherum natürlich, erregt ein Erreger konstant einen Resonater, so wird die Schwingung eingeschränkt, wenn die Erregerfrequenz nicht die Eigenfrequenz ist.
 
\subsection{Beispiele}
Zu den Beispielen gehören
\begin{enumerate}
 \item Brücken, auf welchen marschiert wird 
 \item Federungen
 \item Gläser 
\end{enumerate}
 
\subsection{Experiment}
Die Resonanz kann experimentell aufgezeigt werden, indem zwei Fadenpendel mit einem Faden gekoppelt werden. Wird das eine Fadenpendel ausgelenkt, so erregt dieses das andere Fadenpendel, auch zu schwingen.
 
Weil das erste Fadenpendel von sich aus in seiner Eigenfrequenz schwingt, anhängig von der Fadenlänge, so ist die Energieübertragung maximal wenn beide Fäden gleich lang sind, so dass beide Eigenfrequenzen gleich groß sind.
